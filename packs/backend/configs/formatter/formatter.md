## JavaScript / TypeScript (Prettier)

Install: `npm install --save-dev prettier`

**.prettierrc**
```json
{
  "semi": true,
  "singleQuote": true,
  "trailingComma": "all",
  "printWidth": 100,
  "tabWidth": 2,
  "arrowParens": "always"
}
```

**.prettierignore**
```
dist/
build/
coverage/
*.generated.ts
```

Run: `npx prettier --write .`

## PHP (Laravel Pint / PHP-CS-Fixer)

Install: `composer require laravel/pint --dev`

**pint.json**
```json
{
  "preset": "laravel",
  "rules": {
    "array_syntax": { "syntax": "short" },
    "ordered_imports": { "sort_algorithm": "alpha" },
    "no_unused_imports": true,
    "trailing_comma_in_multiline": true,
    "method_argument_space": { "on_multiline": "ensure_fully_multiline" }
  }
}
```

Run: `./vendor/bin/pint`

## Python (Black + isort)

Install: `pip install black isort`

**pyproject.toml**
```toml
[tool.black]
line-length = 100
target-version = ["py311"]
skip-string-normalization = false

[tool.isort]
profile = "black"
line_length = 100
```

Run: `black . && isort .`

## Go (gofmt / goimports)

Built into the toolchain — no install needed.

Run: `gofmt -w . && goimports -w .`

Add to pre-commit: `gofmt -l . | grep -v vendor | xargs -r gofmt -w`

## Ruby (RuboCop)

Install: `gem install rubocop`

**.rubocop.yml**
```yaml
AllCops:
  TargetRubyVersion: 3.2
  NewCops: enable
  Exclude:
    - 'db/schema.rb'
    - 'bin/**/*'
    - 'vendor/**/*'
Layout/LineLength:
  Max: 120
Style/StringLiterals:
  EnforcedStyle: single_quotes
```

Run: `rubocop -A`
