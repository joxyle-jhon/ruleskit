## JavaScript / TypeScript (ESLint)

Install: `npm install --save-dev eslint @eslint/js globals`

**eslint.config.js**
```js
import js from '@eslint/js';
import globals from 'globals';

export default [
  js.configs.recommended,
  {
    languageOptions: {
      ecmaVersion: 2024,
      sourceType: 'module',
      globals: { ...globals.node },
    },
    rules: {
      'no-console': ['warn', { allow: ['warn', 'error'] }],
      'no-unused-vars': ['error', { argsIgnorePattern: '^_' }],
      'no-eval': 'error',
      'no-implied-eval': 'error',
      'no-return-await': 'error',
      'require-await': 'error',
      'eqeqeq': ['error', 'always'],
      'prefer-const': 'error',
      'no-var': 'error',
    },
  },
];
```

Run: `npx eslint . --fix`

## PHP (PHPStan / Larastan)

Install: `composer require --dev phpstan/phpstan`

**phpstan.neon**
```neon
parameters:
  level: 8
  paths:
    - app
    - tests
  excludePaths:
    - vendor
  checkMissingIterableValueType: false
```

Run: `./vendor/bin/phpstan analyse`

## Python (Flake8 + pylint)

Install: `pip install flake8 pylint`

**.flake8**
```ini
[flake8]
max-line-length = 100
extend-ignore = E203, W503
exclude =
    .git,
    __pycache__,
    dist,
    build,
    .venv
per-file-ignores =
    tests/*: S101
```

Run: `flake8 .`

## Go (golangci-lint)

Install: `go install github.com/golangci/golangci-lint/cmd/golangci-lint@latest`

**.golangci.yml**
```yaml
linters:
  enable:
    - errcheck
    - gosimple
    - govet
    - ineffassign
    - staticcheck
    - unused
    - gosec
    - revive
linters-settings:
  revive:
    rules:
      - name: exported
      - name: var-naming
run:
  timeout: 5m
  skip-dirs:
    - vendor
```

Run: `golangci-lint run`

## Ruby (RuboCop — linting only)

Install: `gem install rubocop rubocop-performance rubocop-rails`

**.rubocop.yml**
```yaml
require:
  - rubocop-performance
  - rubocop-rails
AllCops:
  NewCops: enable
  Exclude:
    - 'db/schema.rb'
    - 'vendor/**/*'
Metrics/MethodLength:
  Max: 20
Metrics/ClassLength:
  Max: 200
```

Run: `rubocop`

## Enforcement

- Run linter in CI on every PR — fail the pipeline on any error
- Run linter as a pre-commit hook (see Husky extra) for fast feedback
- Do not suppress linter errors with inline ignores without a comment explaining why
