# Frontend Excellence

- **HTML Standards**: Use semantic elements (`<main>`, `<header>`, `<article>`, etc.) to improve accessibility (a11y) and SEO.
- **CSS & Design**:
  - Prefer modern CSS layouts (Flexbox, Grid) and responsive designs.
  - Implement smooth transitions and hover effects to make user interfaces feel lively and responsive.
- **JavaScript & Performance**:
  - Optimize JavaScript execution: defer/async non-critical scripts, eliminate blocking main-thread calculations.
  - Apply lazy loading for images (`loading="lazy"`) and components to reduce initial bundle size and speed up page load.
- **Web Vitals**: Always optimize for Core Web Vitals: LCP (Largest Contentful Paint), FID (First Input Delay), and CLS (Cumulative Layout Shift).
